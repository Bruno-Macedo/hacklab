### SMBMAP
- Default

- Enumerate
  - -u USERNAME
  - -r DiskName
    - smbmap -H $target -r DiskName
  - -u USER -H $target -r /ShareNAME/Folder

  - -x COMMANDS
- Download

  - smbmap -u USER -H $target -r /ShareNAME/Folder ---download "share/file.ext"
- Options
  - -H: host
  - -r: path
  - -u: User
  - -p: password
